# Python Function Exercises

This repository contains simple and beginner-friendly Python scripts demonstrating how to define and call functions. Each example focuses on core concepts such as parameters, docstrings, and formatted output.

## Contents

### **example1.py**
Shows how to create a function that accepts a username and prints a personalized message.

### **example2.py**
Demonstrates a function that displays a message about your favorite book, with the title automatically formatted using `.title()`.

## Purpose
These scripts are part of a practice exercise to strengthen foundational Python skills, including:
- Writing functions
- Using docstrings
- Passing arguments
- Formatting strings with f-strings

## How to Run
Clone the repository and run the files using:

```bash
python example1.py
python example2.py
